RawCode Plugin User Guide

The RawCode Wordpress Plugin

  • Description

    Insert complete code (HTML, CSS, and Javascript) directly to Wordpress posts and pages.

    The Raw Code plugin allows code to be inserted directly into posts and pages. By default Wordpress parses content entered into posts and pages and automatically inserts `<p>` and `</p>` paragraph tags around what it deems to be paragraph content. This breaks HTML, CSS, javascript, and other code, preventing it from functioning. Examples of usage include inserting web form code created in another system (email marketing system, CRM system, etc.).

Frequently Asked Questions

  • What code can I use?

    The Raw Code Plugin accepts full HTML, CSS, and Javascript code. PHP cannot be used.

  • Where can code be placed?

    With the Raw Code Plugin, code can be placed into posts, pages, sidebars, headers, footers where site administrators can add and edit content. Simply paste or enter a Replacement Tag code in the content editor field, and publish the content.

    NOTE: Be sure to size content to fit the content area. For example if your site sidebar is 200 pixels wide, then your Raw Code content width should not exceed 200 pixels.
